# CRM Leads Dashboard

## Overview
This dashboard provides a high-level view of CRM lead volume, acquisition channels, and data completeness to support sales and marketing operations.

## Metrics Covered
- Total leads
- Lead volume by source
- Monthly lead trends
- Industry and country distribution
- Contact data completeness

## Data Source
- Cleaned CRM leads dataset from the CRM Data Quality & Cleaning Operations project
